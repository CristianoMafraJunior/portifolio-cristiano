# 🚀 Cristiano Mafra Junior - Portfolio

Professional resume and portfolio built with **React 19**, **Tailwind CSS v4**, and **Motion**.

## 👨‍💻 Sobre o Projeto

Este projeto é um currículo interativo e portfólio profissional desenvolvido para destacar minhas competências como **Desenvolvedor Odoo & Python**. O objetivo foi criar uma experiência web fluida, moderna e funcional, permitindo que recrutadores e parceiros conheçam meu trabalho de forma rápida e elegante.

## ✨ Funcionalidades

- 🌓 **Dark Mode Nativo:** Alternância suave entre temas claro e escuro com persistência de preferência.
- 📄 **Exportação para PDF:** Otimizado para impressão, permitindo gerar um currículo profissional em PDF com um clique.
- 📱 **Totalmente Responsivo:** Design adaptável para qualquer tamanho de tela, de smartphones a monitores ultra-wide.
- 🎭 **Animações Fluidas:** Transições e entradas suaves utilizando a biblioteca `motion`.
- 🛠️ **Foco em Performance:** Construído com as tecnologias mais recentes para garantir carregamento instantâneo.

## 🛠️ Tecnologias Utilizadas

- **Frontend:** [React 19](https://react.dev/)
- **Estilização:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Animações:** [Motion](https://motion.dev/)
- **Ícones:** [Lucide React](https://lucide.dev/)
- **Build Tool:** [Vite](https://vitejs.dev/)

## 🚀 Como Executar

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/CristianoMafraJunior/portfolio.git
   ```

2. **Instale as dependências:**
   ```bash
   npm install
   ```

3. **Inicie o servidor de desenvolvimento:**
   ```bash
   npm run dev
   ```

4. **Acesse no navegador:**
   `http://localhost:3000`
